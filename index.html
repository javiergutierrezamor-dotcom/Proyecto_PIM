1. Introducción

En esta página web vamos a explicar los componentes principales de una placa base (vistos en el módulo de Montaje) y los comandos para usar la terminal de Linux (vistos en el módulo de SOM).

2. Componentes de la Placa Base

La placa base es el elemento clave del ordenador. Aquí tienes una tabla con sus partes principales:

Componente Descripción
Socket Es el lugar donde se conecta la CPU.
Slots RAM Conectores alargados donde se colocan los módulos de memoria RAM.
PCI Express Sirven para conectar tarjetas adicionales, como la tarjeta gráfica o de red.
Chipset Es el circuito integrado que ayuda al procesador a comunicarse con el resto de componentes.
Puertos SATA / M.2 Conexiones dedicadas para los discos HDD y SSD.
Conector ATX Es el puerto grande por donde la fuente de alimentación le da energía a la placa base.

3. Comandos de la Terminal de Linux

Tabla de referencia con los comandos básicos que usamos en el día a día en SOM:

Comando ¿Para qué sirve? Ejemplo
ls Listar los archivos y carpetas del directorio actual. ls -l
cd Cambiar de directorio. cd Documentos
pwd Mostrar la ruta de la carpeta en la que estás metido. pwd
mkdir Crear una carpeta nueva. mkdir trabajos_smr
rm Borrar un archivo. rm apuntes.txt
sudo Ejecutar comandos como administrador. sudo apt update
clear Limpiar la pantalla de la terminal si está muy llena. clear
